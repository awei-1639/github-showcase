# Plan: GitHub Showcase - 渐进增强版本

## Context

用户希望改进现有 GitHub 资源展示网站，要求：
- 更柔和（已完成基础风格）
- 找资源更方便（已完成基础搜索/筛选）
- **新需求**：内容呈现更突出、交互更流畅、探索发现更智能

## 方案：渐进增强（方案 A）

在现有柔和风格基础上，增加特色功能强化内容呈现和探索发现。

---

## 视觉增强

### 配色（保持现有）
- 背景：`#fafafa`
- 卡片：`#ffffff`
- 强调色：`#6366f1`
- 边框：`#e4e4e7`

### 微交互
- 卡片悬浮：轻微上浮（translateY -4px）+ 阴影加深，0.3s ease-out
- 筛选切换：卡片列表 stagger 淡入（50ms 间隔）
- 详情面板：从右侧滑入，400ms spring 动画（damping: 25, stiffness: 200）

---

## 内容呈现功能

### 1. 编辑精选（首页）

首页顶部增加「编辑精选」区块，展示 3-5 个推荐项目。

**UI 布局：**
```
┌────────────────────────────────────────────────────────┐
│  ★  编辑精选                                              │
├────────────────────────────────────────────────────────┤
│  ┌──────────┐  ┌──────────┐  ┌──────────┐              │
│  │ 项目 1    │  │ 项目 2    │  │ 项目 3    │              │
│  │ 精选理由   │  │ 精选理由   │  │ 精选理由   │              │
│  └──────────┘  └──────────┘  └──────────┘              │
└────────────────────────────────────────────────────────┘
```

**展示规则：**
- 从 `repos.json` 中 `featured: true` 的项目选取
- 最多 5 个，优先展示最新添加的
- 每个卡片显示：项目名、描述、一句话精选理由

### 2. 趋势 Tab（新增视图）

新增「趋势」视图入口，展示近期增长最快的项目。

**排序逻辑：**
- 按 Stars 增长比率排序（需要后端数据支持）
- 简化版：直接按 Stars 总数倒序

**UI：**
- Tab 切换：全部 / 趋势 / 精选
- 趋势项目显示「▲」图标和增长数字（如 ▲12%）

## 交互体验

### 流畅动画

**卡片列表过渡：**
```tsx
// Framer Motion stagger 动画
const containerVariants = {
  hidden: { opacity: 0 },
  show: {
    opacity: 1,
    transition: { staggerChildren: 0.05 }
  }
}

const itemVariants = {
  hidden: { opacity: 0, y: 10 },
  show: { opacity: 1, y: 0, transition: { duration: 0.3, ease: "easeOut" } }
}
```

**详情面板动画：**
```tsx
// 从右侧滑入
const panelVariants = {
  hidden: { x: "100%", opacity: 0 },
  show: { x: 0, opacity: 1, transition: { type: "spring", damping: 25, stiffness: 200 } },
  exit: { x: "100%", opacity: 0, transition: { duration: 0.2 } }
}
```

### 悬浮微动效

卡片悬浮时的柔和效果：
```css
.repo-card {
  transition: transform 0.3s ease-out, box-shadow 0.3s ease-out;
}

.repo-card:hover {
  transform: translateY(-4px);
  box-shadow: 0 12px 32px rgba(0,0,0,0.08);
}
```

## 探索发现功能

### 1. 相似项目推荐（详情页）

在 RepoPanel 底部显示 3 个相似项目。

**推荐逻辑：**
1. 同分类 + 相关标签（权重 2:1）
2. 排除当前项目
3. 取前 3 个

**UI：**
```
┌────────────────────────────────────────┐
│ 相似项目                               │
├────────────────────────────────────────┤
│ [小卡片] [小卡片] [小卡片]              │
└────────────────────────────────────────┘
```

### 2. 为你推荐（首页）

首页侧边栏底部显示个性化推荐。

**推荐逻辑：**
- 基于最近搜索词（存储在 localStorage）
- 匹配搜索词相关的项目
- 无搜索历史时显示热门项目

**UI：**
```
┌────────────────────────────────────────┐
│ 为你推荐                               │
├────────────────────────────────────────┤
│ • 项目 A - 相关原因                     │
│ • 项目 B - 相关原因                     │
│ • 项目 C - 相关原因                     │
└────────────────────────────────────────┘
```

## 技术说明

- 使用 Framer Motion 处理所有动画
- 动画时长：300-400ms
- 缓动函数：`ease-out` / `spring`
- 避免：扫描线、闪烁、脉冲等强视觉动画
